# gulp-template

Стартовый шаблон для сборки сайта, основанного на HTML-файлах.

## Инструкция по применению

### Установка модулей

```bash
npm ci
```

### Сборка проекта в папку build

```bash
npm run build
```

### Запуск в режиме разработки

```bash
npm start
```

### Запуск линтеров и валидаторов

```bash
npm test
```
